# Raspberry Pi 3 case for Ender 3 Max
A case for a Raspberry Pi 3 that can be mounted on a Crealtity Ender 3 Max 3D Printer.

## Requirements
### For the case
- Four M2.5x20 screws
- Four M2.5 nuts

### For mounting the case on the printer
- Four M3x10 screws
- Four 6mm T-nuts

### For the camera mount
- One M3x15 screw
- One M3 nut
